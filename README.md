# About
IP generator and checker written in bash. Generate random IPs and scan for ICMP.
    
# Download & Usage
    curl -LO https://files.rmly.dev/ipgen.tar   
    tar xvf ipgen.tar
    cd ipgen
    bash gen -h
   Put your classes in format `xx.xx` in `classes.txt`
